# Gemini Voice Assistant - Голосовой Ассистент для Windows

**Современный голосовой ассистент** с автоматическим распознаванием речи и обработкой текста через AI, разработанный на Python с использованием PySide6.

## 📋 Описание

Voice Assistant — это десктопное приложение для Windows, которое позволяет преобразовывать голосовой ввод в текст с последующей обработкой через Google Gemini AI. Приложение работает в фоновом режиме и активируется по горячим клавишам.

### Основные возможности

- **Распознавание речи** через Faster Whisper (модели small и medium)
- **Обработка текста** через Google Gemini 2.5 Flash/Pro
- **Два режима записи**: обычный и непрерывный (до 15 секунд)
- **Горячие клавиши**: Win+Shift и F1 для быстрой активации
- **Поддержка VLESS VPN** для обхода блокировок
- **Настраиваемый интерфейс** с компактным и развернутым режимами
- **Автозапуск с Windows** и работа в системном трее
- **История записей** с возможностью просмотра и копирования
- **Профили промптов Gemini**: создавайте, копируйте и переключайтесь между разными инструкциями для обработки текста

## 🎯 Особенности

### Режимы работы

1. **Обычный режим**: записывает речь, пока удерживается клавиша
2. **Непрерывный режим**: записывает 15-секундные сегменты автоматически

### Активационные слова

- **"Про"** — использование Gemini 2.5 Pro для сложных задач
- **"Флеш"** — использование Gemini 2.5 Flash для быстрой обработки
- **"Выделить"** — обработка предварительно выделенного текста

## 📦 Установка

### Системные требования

- **OS**: Windows 10/11
- **Python**: 3.8 или выше
- **RAM**: минимум 4 GB (рекомендуется 8 GB)
- **Диск**: ~2 GB свободного места для моделей

### Шаг 2: Установка FFmpeg

FFmpeg необходим для обработки аудио файлов:

1. Скачайте FFmpeg с [официального сайта](https://ffmpeg.org/download.html)
2. Распакуйте архив
3. Добавьте путь к `ffmpeg.exe` в переменную среды PATH

### Шаг 3: ⚠️ ВАЖНО - Установка моделей Faster Whisper

**КРИТИЧЕСКИ ВАЖНО**: Модели Faster Whisper **НЕ ЗАГРУЖАЮТСЯ АВТОМАТИЧЕСКИ**. Их нужно скачать вручную и поместить в папку `whisper_models`.

#### Скачивание моделей:

1. Создайте папку `whisper_models` рядом с исполняемым файлом:

```
D:\Python-app\System\GeminiVoiceAssistant\whisper_models\
```

2. Скачайте модели вручную:
    - **Модель Small**: https://huggingface.co/guillaumekln/faster-whisper-small
    - **Модель Medium**: https://huggingface.co/guillaumekln/faster-whisper-medium
3. **Структура папок должна быть ТОЧНО такой**:

```
whisper_models/
├── faster-whisper-small/
│   ├── config.json
│   ├── model.bin
│   ├── vocabulary.txt
│   └── ...
└── faster-whisper-medium/
    ├── config.json
    ├── model.bin
    ├── vocabulary.txt
    └── ...
```

4. **Проверка**: Убедитесь, что в настройках приложения (вкладка "Основные") модель отображается как активная

**Без правильно установленных моделей приложение НЕ СМОЖЕТ распознавать речь!**

### Шаг 4: Получение API ключа Gemini

1. Перейдите на [Google AI Studio](https://makersuite.google.com/app/apikey)
2. Создайте новый API ключ
3. В приложении откройте **Настройки → Gemini → API ключ**
4. Вставьте полученный ключ

## ⚙️ Конфигурация

### Первичная настройка

1. **Запустите приложение** — появится иконка в системном трее
2. **Откройте настройки** — нажмите на кнопку ▼ в главном окне
3. **Выберите микрофон** — вкладка "Аудио"
4. **Выберите модель Whisper** — вкладка "Основные" (small или medium)
5. **Введите API ключ Gemini** — вкладка "Gemini"
6. **Настройте VPN (опционально)** — вкладка "VPN"

### Настройка VLESS VPN

Если Google Gemini API заблокирован в вашем регионе:

1. Получите VLESS URL от вашего VPN-провайдера
2. Откройте **Настройки → VPN**
3. Включите "Использовать VLESS VPN"
4. Вставьте VLESS URL (формат: `vless://uuid@server:port?...`)
5. Настройте порт SOCKS5 (по умолчанию 10809)
6. Включите "Автоматически подключаться при запуске"

### Горячие клавиши

- **Win + Shift**: Запись в обычном или непрерывном режиме (настраивается)
- **F1**: Запись в обычном или непрерывном режиме (настраивается)

## 🔧 Настройка промпта Gemini

Вы можете настроить поведение Gemini для обработки текста:

1. Откройте **Настройки → Gemini**
2. Выберите профиль промпта из выпадающего списка и при необходимости создайте новый (`+`), переименуйте (`✎`) или удалите (`🗑`) существующий. Текст каждого профиля хранится отдельно и автоматически подставляется в поле ниже.
3. Отредактируйте поле "Промпт для форматирования" — оно всегда отражает выбранный профиль.
4. Настройте активационные слова для специальных режимов

**Как работают профили**:

- При первом запуске создаётся профиль `Default`, в который записывается базовый промпт.
- Текущий профиль хранится в `settings.json` под `gemini_prompts`, а выбор обозначается `gemini_selected_prompt`.
- При переключении профиля текст синхронизируется с полем промпта и сохраняется при вводе.

**Пример промпта**:

```
Ты — эксперт по редактированию речи и текстов, полученных с микрофона.
Твоя задача — преобразовать сырой текст в чистую, отредактированную версию.
Требования:
1. Проверь грамматику, стиль, логику и ясность.
2. Иностранные термины пиши на английском.
3. Выведи только итоговый текст без комментариев.
4. Не изменяй падежи, род и число слов.
```

## 📝 Использование

### Базовый workflow

1. **Нажмите Win+Shift** и начните говорить
2. **Отпустите клавиши** — начнется обработка
3. **Текст автоматически вставится** в активное окно

1. Выделите текст в любом приложении
2. Нажмите **Win+Shift** и скажите: **"выделить [ваша команда]"**
3. Gemini обработает выделенный текст согласно вашей команде

### Использование Pro модели

1. Нажмите **Win+Shift**
2. Скажите: **"про [ваш запрос]"**
3. Запрос будет обработан моделью Gemini 2.5 Pro

## 🐛 Решение проблем

### Приложение не распознает речь

- ✅ Убедитесь, что модели Whisper загружены в папку `whisper_models/`
- ✅ Проверьте правильность структуры папок моделей
- ✅ В настройках выберите корректный микрофон
- ✅ Проверьте уровень громкости в индикаторе

### Ошибка Gemini API

- ✅ Проверьте правильность API ключа в настройках
- ✅ Если API заблокирован — настройте VLESS VPN
- ✅ Проверьте подключение к интернету
- ✅ Убедитесь, что порт прокси доступен

### VPN не подключается

- ✅ Проверьте корректность VLESS URL
- ✅ Убедитесь, что `xray.exe` присутствует в папке
- ✅ Проверьте доступность порта SOCKS5
- ✅ Просмотрите логи приложения

### Низкое качество распознавания

- ✅ Используйте модель **medium** вместо small
- ✅ Улучшите качество микрофона
- ✅ Говорите четко и ближе к микрофону
- ✅ Включите "Предупреждать о тихой записи" в настройках
