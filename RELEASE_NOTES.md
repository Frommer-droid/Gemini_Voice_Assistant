## Версия 1.4.0 (2025-11-21)

*Голосовое управление программами и апплетами Windows.*

### ✨ Новые возможности

- **Запуск программ голосом**: команда «Запусти [название]» открывает программы, апплеты Windows и системные утилиты.
- **Умное распознавание команд**: встроенный словарь популярных программ (калькулятор, блокнот, панель управления, браузеры и др.) обеспечивает мгновенный запуск.
- **Gemini AI для неизвестных программ**: если команда не найдена в словаре, Gemini определяет нужную программу по описанию (например, «программа для рисования» → Paint).
- **Защита от опасных команд**: автоматическое обнаружение потенциально опасных операций (удаление файлов, форматирование, изменение реестра) с обязательным диалогом подтверждения.
- **Полное логирование**: все выполненные команды записываются в лог с отметками о подтверждении или отклонении.

### 🛠 Исправления

- Упрощена система триггеров голосовых команд: теперь только формы слова «открой» используются для сайтов, а формы слова «запусти» — для программ. Это устраняет конфликты при распознавании команд.
- Исправлена проблема, из-за которой окно не сворачивалось обратно в компактный режим после выполнения команд запуска программ и открытия сайтов. Теперь окно автоматически минимизируется, как и после диктовки.

### ⚡ Улучшения

- Добавлен словарь из 40+ популярных Windows команд и программ для мгновенного запуска без обращения к Gemini.
- Реализован механизм обнаружения опасных команд с использованием регулярных выражений для паттернов (del, format, reg delete и т.д.).

## Версия 1.3.0 (2025-11-21)

### ✨ Новые возможности
