# Специфика промпта для нормализации запроса поиска через Gemini

## Цель
Приводить распознанную фразу с триггером «найди (папку|файл|...) ... на диске ...» к строгой структуре JSON, чтобы модуль поиска мог построить regex/прямой запрос для es.exe.

## Правила нормализации
1) Триггер: команда должна начинаться с формы «найди/найдите/найти/найдём/найдем» (case-insensitive).
2) Тип объекта:
   - «папка/каталог/директория» → `target_type = folder`
   - «файл/документ» → `target_type = file`
   - если тип не указан → по умолчанию `folder`
3) Имя (`name`):
   - Исправить опечатки, сохранить слова в исходном порядке.
   - Если пользователь явно сказал «<N> слово/слова и все на английском/русском/...», или задал разделители («нижнее подчеркивание», «слить слова», «пробел»), нужно вернуть `name` ровно в указанной форме.
   - Удалить служебные части («папку», «на диске ...»).
4) Диск (`drive`):
   - Если сказано «на диске х» → вернуть букву в нижнем регистре, например `d`.
   - Если диска нет → `all`.
5) Язык/формат:
   - Фразы вида «два слова и оба на английском языке», «три слова на русском» использовать как инструкцию по виду имени. Например: «youtube видео два слова и оба на английском языке» → `name: "youtube video"`.
6) Вывод: один объект JSON одной строкой, без текста вокруг:
```
{"trigger":"найди","target_type":"folder","name":"youtube video","drive":"d"}
```

## Примеры
- Ввод: `найди папку youtube видео два слова и оба на английском языке на диске д`
  Вывод: `{"trigger":"найди","target_type":"folder","name":"youtube video","drive":"d"}`
- Ввод: `найди папку откудо береться оптимязм на диске д`
  Вывод: `{"trigger":"найди","target_type":"folder","name":"откуда берётся оптимизм","drive":"d"}`
- Ввод: `найди папку PortableSoft`
  Вывод: `{"trigger":"найдём","target_type":"folder","name":"portable soft","drive":"all"}`
- Ввод: `найди папку на диске d два ноля потом нижнее подчеркивание и слово развитие`
  Вывод: `{"trigger":"найди","target_type":"folder","name":"00_Развитие","drive":"d"}`
- Ввод: `найди папку на диске d мои сайты, только английскими словами и разделенные нижним подчеркиванием`
  Вывод: `{"trigger":"найди","target_type":"folder","name":"my_sites","drive":"d"}`
